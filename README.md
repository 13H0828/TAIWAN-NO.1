# TAIWAN-NO.1

## Concept Development 開發構想
- 爲了慶祝中華隊棒球奪得冠軍，所以我們做了一個小游戲來回顧我們奪冠的歷程

## Gameplay 玩法
- 鏡頭1和鏡頭2會形成一個好球帶的範圍，玩家可在鏡頭1后方投球，當球經過鏡頭1和鏡頭2時候會判斷為好球

## Implementation Resources 運用資源
- Python
- Ubuntu
- Chatgpt
- Opencv

## Ubuntu Setup 虛擬機設定
